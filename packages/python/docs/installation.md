# Installation

Base package:

```bash
pip install olloweditor
```

Framework extras:

```bash
pip install "olloweditor[django]"
pip install "olloweditor[drf]"
pip install "olloweditor[flask]"
pip install "olloweditor[fastapi]"
pip install "olloweditor[all]"
```

`olloweditor` packages static browser assets and optional framework integrations. Base installation does not install Django, DRF, Flask, or FastAPI.
